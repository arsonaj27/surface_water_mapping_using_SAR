Overview

This project explores monthly surface water dynamics using Synthetic Aperture Radar (SAR) imagery. The workflow includes data acquisition, preprocessing, thresholding (e.g., Otsu), and validation, leading to interpretable maps and time‑series insights.

View Results

Objectives

  • Map surface water extent consistently over time.
  • Use SAR images to reduce cloud-related gaps.
  • Build a transparent, reproducible workflow.

Highlights

  • Fully open-source and reproducible.
  • Ready for academic reports and demos.
  • Interactive map & plots (Leaflet + Plotly).
