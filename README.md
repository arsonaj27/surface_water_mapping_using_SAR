# Surface Water Mapping Using SAR — Website

This repository contains a ready-to-host static website for GitHub Pages.

## How to publish (Project Pages)
1) Commit the `/docs` folder (already provided).
2) Go to **Settings → Pages**.
3) In **Build and deployment**, set:
   - **Source**: *Deploy from a branch*
   - **Branch**: `main`
   - **Folder**: `/docs`
4) Save. Your site will be available at:
   `https://<username>.github.io/<repo-name>/`

Replace placeholder images/plots with your actual outputs.
